# HC Skills for Claude Code

Search, shortlist, and engage workforce solutions on [Human Cloud](https://humancloud.com) — directly from your terminal.

## Quick Start

```bash
# 1. Install
git clone https://github.com/Real-Human-Cloud/hc-skills.git ~/.claude/skills/hc

# 2. Add your API key (get one at https://app.humancloud.com/my-cloud/integrations)
echo "YOUR_API_KEY" > ~/.hc-api-key

# 3. Use it
claude /hc find UX research firms in North America
```

## Commands

| Command | Description |
|---------|-------------|
| `/hc [query]` | Search ranked workforce solutions, shortlist, send briefs or RFPs |
| `/hc-status` | View active shortlists, briefs, and RFPs |

## What You Can Do

- **Search** 500+ ranked workforce solutions with natural language
- **Shortlist** solutions for comparison
- **Send briefs** to get custom proposals
- **Create RFPs** with structured questions

## Requirements

- [Claude Code](https://claude.ai/claude-code) CLI
- Human Cloud account on the **Max** plan
- API key from [Integrations](https://app.humancloud.com/my-cloud/integrations)
